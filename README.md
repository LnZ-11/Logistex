# 📦 Logistex

**Logistex** is a high-performance inventory and sales management solution designed specifically for local deployments. Built with a modern tech stack, it provides a fluid, secure, and robust experience for businesses looking to digitize their operations without relying on cloud infrastructure.

---

## 🚀 Tech Stack

The project leverages cutting-edge technologies to ensure type-safety, maintainability, and performance:

- **Framework:** [Next.js 14+](https://nextjs.org/) (App Router)
- **Language:** TypeScript
- **Authentication:** [Lucia Auth](https://lucia-auth.com/) (Local session management)
- **Database:** [Prisma](https://www.prisma.io/) (ORM) with SQLite/PostgreSQL
- **Validation:** [Zod](https://zod.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [Shadcn/UI](https://ui.shadcn.com/)
- **State Management:** [Zustand](https://github.com/pmndrs/zustand)
- **Icons:** [Lucide-React](https://lucide.dev/)

---

## ✨ Key Features

- 🔐 **Secure Authentication:** Robust session management via Lucia Auth, optimized for offline use.
- 📊 **Inventory Management:** Real-time tracking, editing, and monitoring of stock levels.
- 💰 **Sales Tracking:** Intuitive interface for recording transactions and managing sales data.
- 📄 **Reporting:** Simplified visualization of stock movements and business performance.
- 📱 **Responsive Design:** Fully optimized for desktops, tablets, and POS systems.
